On Includes

Banyak bahasa pemrograman dan file komputer lainnya memiliki arahan, sering disebut include (terkadang salin atau impor), yang menyebabkan konten file kedua dimasukkan ke dalam file asli. File yang disertakan ini disebut copybook atau file header. Mereka sering digunakan untuk menentukan tata letak fisik data program, potongan kode prosedural dan/atau deklarasi penerusan sambil mempromosikan enkapsulasi dan penggunaan kembali kode.
